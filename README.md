## 🚃 개발 기술
### React
- 컴포넌트화를 통해 유지보수와 재사용성 고려
- Virtual DOM을 사용하여 실제 DOM 조작을 최소화. 성능 최적화 및 빠른 UI 업데이트 가능
- 다양한 라이브러리와 도구를 함께 사용할 수 있으며, React Router를 사용해 복잡한 라우팅 관리하고 Redux를 통한 상태관리 가능
- Hooks를 통해 보다 간결하고 이해하기 쉬운 코드 작성 가능
